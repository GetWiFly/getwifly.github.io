Getting Started

Welcome to the official documentation. Follow the steps below to integrate the library into your environment and begin development.

1

Installation

Download our app by the latest version to get started immediately.

2

Extract Files

To get your Wi-Fi Sign-In page up and working, we have a folder full of a few files you need to extract in order to continue.

3

Get your WiFly ID

Sign up using a username, email and password. Your Wifly ID will look like a creative password. For example:

Text 
ChillHorizon2957

Simple, right? That's all the ID will be.

WARNING: Your WiFly ID is a private credential. Never share it in public screenshots, GitHub repositories, or client-side code that isn't obfuscated.
Note: We might change how the WiFly ID's will look like soon, but these will be secured in your dashboard, and you only get to access it using your password.
4

Time to fly!

Your in! You can now start customizing!
