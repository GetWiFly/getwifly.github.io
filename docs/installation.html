Installation Help

To get the best experience using our app, check out what versions are good for your device.

Platform Recommended Version Status
Windows v2.4.0 (Stable) ● Optimized
Apple macOS v2.3.8 (Silicon/Intel) ● Optimized
Apple iOS v2.1.5 (Lite) ● Optimized
Android v2.1.2 (Standard) ● Optimized
Linux v1.9.0 (CLI + GUI) ● Supported
Not seeing your version? If your specific OS version isn't listed, try the Legacy (v1.2.0) build. It has higher compatibility but fewer "Fly-Fast" features.
Security Note: Always verify the checksum of your download to ensure you're getting the official WiFly package.
